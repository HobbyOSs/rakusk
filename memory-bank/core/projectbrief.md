# Project Brief: rakusk

Rakuを基盤とした、自作OSブートストラップ用x86アセンブラ。

## Core Concept
- AI（Cline）が命令定義を生成し、Rakuがそれを読み込んで自己増殖するサイクルを構築する。
- 「完璧なx86アセンブラ」ではなく、「自分のOSをブートさせるための専用コンパイラ」を目指す。

## Goals
- [x] `MOV` 命令の基本実装（レジスタ・即値）
- [x] ModR/M バイト生成の抽象化エンジン構築
- [x] `ndisasm` を利用した自動テストパイプラインの構築
- [x] 512バイトのブートセクタ（boot.bin）の出力
- [x] Day 01 〜 Day 05 (Harib02i) までの全テストパス
- [x] **Mission: Day 06 Suite (Harib03e) 以降のアセンブラ機能完遂**
- [x] C言語連携に必要な呼び出し規約、スタック操作の完全サポート
- [x] **Day 25 までの全 OS サンプルコードのバイナリ一致達成**
