# Active Context

## 最近の変更点
- `gosk` の `pass1` 機能を `lib/Rakusk/Pass1.rakumod` に移植。
    - 各種命令のサイズ計算ハンドラ（`MOV`, `RET`, `INT`, `IN/OUT`, `JMP/CALL`, 算術・論理演算, `PUSH/POP`）を実装。
    - パラメータなし命令のサイズ推定例外に対応。
    - `GLOBAL`, `EXTERN` 疑似命令のサポートを追加。
- ModR/M 計算エンジンの実装完了。レジスタ間転送 (`reg-reg`) をサポート。
- ニーモニックが複数の命令形式（バリアント）を持つ場合、オペランドに応じて最適な定義を選択する機能を `Rakusk::Grammar` に実装。
- `Test::Property` の考え方を取り入れた ModR/M バイト生成の検証用ランダムテストを導入。

## 現在の焦点
- FAR ジャンプ（セグメント付きアドレス）のサポートに向けた AST の拡張。
- メモリオペランドを伴うアドレッシングモードの ModR/M 生成（SIB対応含む）。

## 次のステップ
- **FAR ジャンプの AST 対応**: `SegmentedAddress` ノードの定義とパース、パス1でのサイズ計算。
- **メモリオペランド対応**: ModR/M 演算エンジンを拡張し、SIB バイトやディスプレースメントを扱うロジックを実装。
- **Pass 2 の精緻化**: 解決済みシンボルを用いた最終的なバイナリ生成。
