# Product Context - rakusk

## 製品の目的
`rakusk` は、Rakuプログラミング言語を使用して実装された、自作OSブートストラップ用のx86アセンブラです。

## 解決する問題
- 汎用アセンブラ（NASM等）の複雑な学習曲線を避け、OS自作に必要な機能に特化した軽量な環境を提供。
- AI（Cline）による自動的な命令追加と検証を可能にすることで、開発速度を向上させる。
- ビット計算やバイナリ構造（ModR/M, ブートセクタ構成）の理解を深める。

## 動作方法
1. **記述**: Rakuの `Grammar` を通じてx86アセンブリソースをパース。
2. **変換**: `Actions` クラスが機械語のバイト列（`Buf`）を生成。
3. **検証**: 生成されたバイナリを `ndisasm` を用いて自動的に逆アセンブルし、正当性をチェック。
4. **出力**: 最終的に512バイトのブートセクタ画像（`boot.bin`）等を出力。
